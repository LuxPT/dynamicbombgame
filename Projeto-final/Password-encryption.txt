Os leds na caixa representam um número de base 3.
Ou seja, suponhamos que temos a primeira e a ultima led acesa.

Irei utilizar "O" para representar um led acesso e "0" para representar um led apagado.

O 0 0 - > 3^2 + 0 + 0 = 9;
O 0 O - > 3^2 + 0 + 3^1 = 9 + 1 = 10;
0 O 0 - > 0 + 3^1 + 0 = 3;
O O O - > 3^2 + 3^1 + 3^0 = 9 + 3 + 1 = 13;

Ou seja no fundo, onde cada 1 representa um led aceso e um 0 um apagado:

1 0 0 - > 9

1 0 1 - > 10

0 1 0 - > 3

1 1 1 - > 13
